Initial release of `@aubron/skill-tools` — validate and release Claude Agent
Skills from the package factory. Provides the `aubron-skill` CLI (`validate` a
skill package; `sync-marketplace` to regenerate the root marketplace.json from
the repo's skill packages, with hybrid `git-subdir`/`npm` sources) plus the
underlying library functions.
